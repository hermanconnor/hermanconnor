# Hi there, I’m Herman 👋
🔬 **`From Lab Bench to Code Base`** 💻

I’m a software developer with a background in biological sciences and medical laboratory technology, with experience as a research technician in a transplant immunobiology lab. My work in the lab sharpened my ability to analyze complex data, think critically, and collaborate across disciplines.

Today, I build full-stack applications using modern technologies like JavaScript (Node.js, React) and Python, with a focus on creating reliable, scalable solutions. I’m continually expanding my skills in system design, cloud architecture, and DevOps practices.

I’m driven by the intersection of healthcare, data, and software engineering, and I’m passionate about developing tools that turn complex data into meaningful, real-world impact.
